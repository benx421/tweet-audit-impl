# Tweet Audit | Java Implementation

Java implementation of the Tweet Audit tool.

## Prerequisites

- Java 25
- Maven 3.9+

## Build & Run

```bash
# Build
mvn clean package

# Run
java -jar target/tweet-audit.jar
```

## Development

```bash
# Test
mvn test

# Code style check
mvn checkstyle:check
```
